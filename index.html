FedWatch — โอกาสที่ Fed จะปรับดอกเบี้ยในแต่ละการประชุม

ความน่าจะเป็นของกรอบดอกเบี้ยปลายทาง อิงจาก 30-Day Fed Funds Futures แถบสีบอกว่าตลาด "คง" (เทา) / "ลด" (เขียว) / "ขึ้น" (แดง) เทียบกับปัจจุบัน

การแจ้งเตือน (Alerts)

เตือนเมื่อโอกาส "ลด" หรือ "ขึ้น" ดอกเบี้ยในรอบที่เลือก ≥ เกณฑ์ที่กำหนด — เงื่อนไขถูกบันทึกไว้ในเครื่อง (localStorage) และตรวจทุกครั้งที่ข้อมูลอัปเดต

≥ %

ประวัติการเปลี่ยนแปลงโอกาส

โอกาส ลด/คง/ขึ้น ของการประชุมที่เลือก เปลี่ยนแปลงอย่างไรเมื่อเวลาผ่านไป (สะสมวันละจุดเมื่อรันผ่าน backend)

FedWatch แม่นแค่ไหน — เทียบกับผลจริงย้อนหลัง

แต่ละแถวคือการประชุมในอดีต: แถบ = ความน่าจะเป็นที่ตลาดให้ "1 วันก่อนประชุม" · กรอบขาว = ผลที่ เกิดจริง · ป้ายขวา = ทายถูกไหม และให้โอกาสกับผลจริงกี่ %

Dot Plot — มุมมองดอกเบี้ยของสมาชิก FOMC

แต่ละจุดคือการคาดการณ์ของสมาชิก FOMC 1 คน ต่อระดับดอกเบี้ย ณ สิ้นปีนั้นๆ (จาก SEP ) — เส้นประคือค่ามัธยฐาน (median)

Fed vs ตลาด — เทียบ Dot Plot กับที่ตลาดคาด

แท่งฟ้า = ช่วงการคาดการณ์ของ FOMC (เส้นเหลือง = median) · เพชรเขียว = อัตราที่ตลาดคาด ณ สิ้นปี (จาก FedWatch) — ตัวเลขกำกับคือ "ส่วนต่าง" เป็น bps (บวก = Fed มองดอกเบี้ยสูงกว่าตลาด/เหยี่ยวกว่า)

เส้นทางอัตราดอกเบี้ย (Rate Path)

เส้นทึบ = อัตราจริงในอดีต · เส้นประ = เส้นทางที่ตลาดคาด (อิง FedWatch)
